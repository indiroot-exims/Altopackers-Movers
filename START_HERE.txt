🔧 FORM SETUP:

Contact form is ready but needs an email service:

Option 1: Formspree (Easiest)
  1. Go to formspree.io
  2. Sign up (free)
  3. Create form, copy ID
  4. In contact.html, update form action
  5. Done! Emails work

Option 2: EmailJS (Client-side, no backend)
  1. Go to emailjs.com
  2. Sign up (free)
  3. Get credentials
  4. Update script.js with credentials
  5. Done! Emails work

See INSTALLATION.md for more details.
