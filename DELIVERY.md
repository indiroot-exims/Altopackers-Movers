### Update Contact Info
```html
<!-- Search for these and update -->
+91 8000709840               → Your phone
221 - Aristo, Kabir Road     → Your address
info@altopackers.com         → Your email
```

### Change Colors
```css
:root {
  --color-dark: #0B1120;        /* Main navy */
  --color-accent: #D4A94E;      /* Gold */
  --color-light: #F8F7F4;       /* Off-white */
}
```

### Replace Hero Video
```html
<source src="YOUR_VIDEO_URL.mp4" type="video/mp4">
```

### Set Up Contact Form Email
Choose one:
- **Formspree** (easiest) - Update form action
- **EmailJS** (client-side) - Add credentials to script.js
- **Getform** (similar to Formspree)
